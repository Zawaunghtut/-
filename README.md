
Vagga Yatra Calculator (ဝဂ္ဂ ယတြာ တွက်ချက်မှုစနစ်)
Vagga Yatra Calculator သည် သရောဒယဗေဒင်ပညာ (Swarodaya Astrology) ကို အခြေခံ၍ လူအမည်၊ ဘုရားအမည် နှင့် လှူဖွယ်ပစ္စည်းများအတွက် သင့်တော်သော ဝဂ္ဂယတြာများကို တိကျစွာ တွက်ချက်ပေးသည့် System တစ်ခု ဖြစ်ပါသည်။
🌟 Key Features (အဓိက လုပ်ဆောင်ချက်များ)
 * Swarodaya Astrology Logic: ရှေးဟောင်း သရောဒယဗေဒင်ကျမ်းလာ ဝဂ္ဂတွက်ချက်နည်းစနစ်များကို Digital စနစ်သို့ ပြောင်းလဲထားခြင်း။
 * Security Feature: Random Code နှင့် Password စနစ် ပါဝင်သောကြောင့် ခွင့်ပြုချက်ရှိသူများသာ အသုံးပြုနိုင်ရန် ကန့်သတ်ထားနိုင်သည်။
 * Myanmar Name Input: မြန်မာအမည်များကို တိုက်ရိုက်ရိုက်ထည့်၍ တွက်ချက်နိုင်ခြင်း။
 * Elegant UI: ရိုးရာနှင့် ခေတ်မီဒီဇိုင်းကို ပေါင်းစပ်ထားသော Luxury Dark Theme အသုံးပြုထားခြင်း။
🚀 How It Works (အလုပ်လုပ်ပုံ)
 * User မှ System ထဲသို့ ဝင်ရောက်ရန်အတွက် ပြသထားသော Random Code ကို အသုံးပြု၍ Admin ထံမှ Password ရယူရပါမည်။
 * Password အမှန်ထည့်သွင်းပြီးနောက် မိမိတွက်ချက်လိုသော လူအမည် (သို့မဟုတ်) ဘုရားအမည် ကို မြန်မာလို ရိုက်ထည့်ပါ။
 * System မှ သတ်မှတ်ထားသော ဝဂ္ဂနည်းစနစ်များအရ အကျိုးပေးမည့် ယတြာနှင့် အစီအမံများကို ထုတ်ပေးမည်ဖြစ်ပါသည်။
🛠️ Tech Stack (အသုံးပြုထားသော နည်းပညာများ)
 * Frontend: HTML5, CSS3 (Custom Styles), JavaScript
 * Backend Logic: Complex Swarodaya calculation algorithms
 * Visuals: Premium Myanmar Typography & Golden Graphics
